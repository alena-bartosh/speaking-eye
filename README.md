# Speaking Eye

### System Dependencies
```sh
sudo apt-get install \
    python3-gi \
    gir1.2-gtk-3.0 \
    gir1.2-wnck-3.0 
```

### One Command Setup
```sh
git clone https://github.com/alena-bartosh/speaking-eye.git && \
cd speaking-eye && \
python3 -m venv .env && \
source .env/bin/activate && \
pip install -r requirements.txt
```
